웹사이트 로딩

스크립트 오류 창이 뜰 때 어떻게 해결하나요?

우선 인터넷 익스플로러를 열고 메뉴 → 도구 → 인터넷 옵션을 들어가 줍니다. 익스플로러 버전 11은 이렇게 우측 상단에 보시면 톱니바퀴가 있는데 이걸 클릭하시면 바로 인터넷 옵션에 들어가실 수 있습니다.

여기서 [고급] 탭으로 들어가서 [원대대로(S)] 를 눌러주게 되면 아래와 같은 팝업창이 뜨게 됩니다.

팝업창에 뜨면 [다시 설정]을 눌러주시면 됩니다.
자, 여기서 보시면 [개인 설정 삭제]라는 항목이 보이시나요? 저 항목을 체크하게 되면 열어보셨던 웹페이지 목록과 자동로그인 했던 기록들이 다 사라기지 때문에 저런 목록들이 삭제 되는 걸 원치 않으신다면 체크를 하지 마시고 다시 설정을 눌러주시면 됩니다.

▶ 더 간단한 TIP
모든 페이지가 아니라 특정한 페이지에서만 이러한 오류가 발생한다면 다음과 같은 방법으로도 해결할 수 있습니다.
아까 [고급]탭에서 이렇게 설정을 해주시면 됩니다. 아래와 같이 두 항목을 찾아서 체크 해제를 해주시면 되겠습니다!
이러면 이제 짜증나게 뜨던 스크립트 오류를 해결하실 수 있으실 겁니다

사이트의 사진이 보이지 않을 때는 어떻게 하나요?

먼저, Internet Explorer 10에서 해당 사이트가 호환성 문제가 있지 않은지 확인해 보시기 바랍니다.
경우에 따라 방문하고 있는 웹 사이트가 예상대로 표시되지 않을 수 있습니다. 이미지가 표시되지 않고 메뉴의 위치가 적절하지 않으며 텍스트 상자가 뒤섞여 있을 수 있습니다. 이러한 현상은 Internet Explorer와 현재 사이트 간의 호환성 문제로 인해 발생할 수 있습니다.

아래 링크를 통하여 호환성 보기 기능도 참고해 보시고, 동일 시 사이트 지원센터를 통하여 호환성 여부를 확인해 보시기 바랍니다.

작업 방법 1. 호환성 보기를 사용하여 사이트 표시 문제 해결하기
http://windows.microsoft.com/ko-kr/internet-explorer/use-compatibility-view#ie=ie-10

작업 방법 2. GPU렌더링 사용해제
Internet Explorer 10은 그래픽및 텍스트 렌더링을 CPU에서 GPU로옮겨 웹 페이지의그래픽 성능과 렌더링을가속화합니다. 해당 기능으로인해 다른 하드웨어장치와의 문제가 발생되거나 정상적이지 않은경우 GPU 렌더링을 해제하여 문제점을 확인하여 봅니다

제어판 - 인터넷 옵션 - 고급 탭을 클릭한 후 가속그래픽 섹션으로이동합니다.
GPU렌더링 대신소프트웨어 렌더링사용 확인란을클릭합니다.
적용, 확인을클릭합니다.
모든 Internet Explorer 창을 닫고 다시 Internet Explorer 을시작 합니다.

로그인

사이트 로그인이 안 될 경우 어떻게 하나요?

사이트는 인터넷 익스플로러 9버전부터 원활한 이용이 가능하며
로그인이 정상적으로 되지 않을 경우, 아래의 방법대로 1차 조치를 부탁 드립니다

1. 인터넷 익스플로러 사용자
1) 인터넷 실행 후 상단의 메뉴 중 [도구] → [인터넷 옵션] 클릭
※ [도구] 메뉴가 보이지 않을 경우 키보드의 [Alt] 키를 입력하면 메뉴가 나타납니다.
2) [고급] 탭에서 [원래대로] 버튼 클릭
3) [개인 설정 삭제] V 체크 후 [다시 설정] 클릭

2. 크롬 사용자
1) 크롬 브라우저 실행 후 키보드의 [ctrl + shift + delete] 키 동시 입력
2) 다음 항목 삭제 [전체] → [체크박스] 전체 선택 → [인터넷 사용 기록 삭제] 클릭

아이디/비밀번호가 기억나지 않아요

아이디와 비밀번호는 회원님이 입력하신 이메일을 통해 직접 확인하셔야 하며, 이메일도 기억나지 않으신 경우 문의 전화 주시면 안내해 드리겠습니다 *고객센터를 통한 비밀번호 등의 개인 정보 조회/변경 은 불가능 합니다.

(단, 여러가지 방법으로도 찾지 못하신 경우 임시 비밀번호가 발급됩니다.)

회원 가입을 하려는데 '이미 가입 된 회원입니다'라고 나옵니다

회원님의 인증정보가 이미 등록되어 있는 경우는 다음 2가지로 볼 수 있습니다.

1. 과거에 회원으로 가입하신 후에 가입사실을 잊으신 경우
2. 타인이 회원님의 정보를 이용하여 가입한 경우

먼저 로그인 페이지에서 회원가입 유무를 확인하세요.
과거에 본인이 가입한 경우라면 ID,비밀번호 문의를 이용하여 다시 이용하실 수 있습니다.

만약, 정보를 확인하지 못하시거나 전혀 모르는 타인에 의하여 가입이 되어 있는 경우 고객센타에 고객문의 (02-2008-5200) 및 사이버 수사대에 신고하여 주시기 바랍니다.
